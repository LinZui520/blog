### 搭建属于自己的网站😎

边学习边搭建自己的个人网站

#### 部署 

```sh
$ git clone git@github.com:LinZui520/website.git
```

阅读```./nginx```文件夹下的的```README.md```文件 然后

```website```启动🫡

```sh
$ docker-compose build
$ docker-compose up -d
```
